# op-to-do
My solution for the [Odin Project's To-Do List Excercise](https://www.theodinproject.com/lessons/node-path-javascript-todo-list)

## Minimum Requirments
- [ ] APP: Create task with the following properties:
    - [ ] "title" 
    - [ ] "description"
    - [ ] "dueDate"
    - [ ] "priority"
    - [ ] "isComplete"
- [ ] APP: Choose which project a task is saved to.
- [ ] APP: Delete a task.
    - [ ] Have a default project that all tasks are saved to unless the user creates a new project.
- [ ] DOM: View all projects in one page.
- [ ] DOM: View all tasks in a project.
- [ ] DOM: Expaned view of a single task.
    - [ ] Edit task properties.
    - [ ] Delete a task.
- [ ] APP: Save tasks to local storage.
